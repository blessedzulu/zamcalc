## ğŸ“¦ Installation

To get started with local development:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/blessedzulu/zamcalc.git
   cd zamcalc
   ```

2. **Install dependencies:**

   ```bash
   composer install
   npm install
   ```

3. **Environment Setup:**

   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Run Migrations:**

   ```bash
   php artisan migrate
   ```

5. **Start Dev Server:**
   ```bash
   npm run dev
   php artisan serve
   ```
